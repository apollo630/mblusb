## Dependencies

- [libusb-1.0](https://libusb.info/)
  - Arch: `pacman -S libusb`
  - Debian: `apt install libusb-1.0-0-dev`
  - MacOS: `brew install libusb`


## Build

```
cd build/
cmake ../
make
```
